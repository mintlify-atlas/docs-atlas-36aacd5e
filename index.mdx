## Get Up and Running

<Steps>
  <Step title="Install Prerequisites">
    Install **Visual Studio 2022** (or later) with the **.NET Desktop Development** workload and ensure the **.NET 8 SDK** is available.
  </Step>
  <Step title="Clone & Open the Solution">
    Clone the repository and open `ExploradorDeArchivos.sln` in Visual Studio.
  </Step>
  <Step title="Restore NuGet Packages">
    Visual Studio will automatically restore all NuGet dependencies (NAudio, LibVLCSharp, TagLibSharp, etc.) on first build.
  </Step>
  <Step title="Run the Application">
    Press **F5** or click **Start** to launch. The file explorer opens to your Documents folder by default.
  </Step>
</Steps>

<Note>
  Explorador de Archivos targets **Windows only** (`net8.0-windows`) and requires Windows Forms support. It will not compile or run on macOS or Linux.
</Note>
